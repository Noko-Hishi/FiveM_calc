# 🛒 FiveM ショップ電卓

FiveM サーバー向けの **ショップ料金計算ツール**です。  
商品をカートに追加して合計金額を素早く計算・コピーできます。

---

## 📋 機能一覧

- **商品管理** — 商品名・価格・入力方式・カテゴリを自由に設定
- **カテゴリ分類** — タブ切り替えで商品を素早く絞り込み
- **2種類の入力方式** — 数量入力（＋/−ボタン）または ON/OFF トグル
- **値引き機能** — 固定額・割合（%）どちらでも対応
- **金額コピー** — 合計金額を数字のみクリップボードにコピー
- **設定の保存/読込** — JSON ファイルで複数のショップ設定を管理

### メイン画面

```
┌─────────────────────────────────────┬──────────────┐
│  ショップ名     [設定を読込] [設定を保存] [商品設定] │
├──────────────────────────┬──────────────────────────┤
│ すべて  カテゴリA  カテゴリB  │  カート               │
│                          │                          │
│  商品名          ¥1,000  │  商品名    x2   ¥2,000  │
│  商品名          ¥500    │  商品名    ON   ¥500    │
│  ...                     │                          │
│                          │  ────────────────────── │
│                          │  小計  ¥2,500            │
│                          │  ┌ 値引き ─────────────┐ │
│                          │  │ ○固定額 ○割合(%)   │ │
│                          │  │ ¥______ [クリア]   │ │
│                          │  └───────────────────── │
│                          │  合計金額                │
│                          │  ¥2,500                  │
│                          │  [  金額をコピー  ]      │
│                          │  [    リセット    ]      │
└──────────────────────────┴──────────────────────────┘
```

### 各エリアの操作

| エリア | 操作 |
|--------|------|
| **商品一覧（左）** | カテゴリタブで絞り込み、商品のボタンで数量を調整 |
| **カート（右）** | 追加した商品の一覧。数量変更・削除が可能 |
| **値引き** | 固定額か割合を選んで金額を入力 |
| **金額をコピー** | 合計の数値のみをクリップボードへコピー |
| **リセット** | カート・値引きをすべてクリア（確認あり） |

## ⚙️ 商品設定

ヘッダー右上の **[商品設定]** ボタンから開きます。

### カテゴリ管理

1. カテゴリ名を入力して **[＋ 追加]** をクリック
2. リストで選択して **[✏ 更新]** で名前を変更
3. **[▲ 上へ] / [▼ 下へ]** で並び順を変更
4. **[🗑 削除]** でカテゴリを削除（そのカテゴリの商品は「未分類」になります）

### 商品管理

| 項目 | 説明 |
|------|------|
| **商品名** | カードに表示される名前 |
| **価格（円）** | 整数で入力 |
| **入力方式** | `数量入力（＋/−）` または `ON/OFFトグル` |
| **カテゴリ** | 所属させるカテゴリ（未分類も可） |

1. 各項目を入力して **[＋ 追加]** をクリック
2. リストで選択 → 編集 → **[✏ 更新]** で変更を反映
3. **[▲ 上へ] / [▼ 下へ]** で並び順を変更

## 💾 設定ファイルについて

設定（ショップ名・カテゴリ・商品）は `config.json` に自動保存されます。  
アプリと同じフォルダに置いてください。

### 複数ショップの運用

- **[設定を保存]** → 任意の場所に `.json` として書き出し
- **[設定を読込]** → 保存した `.json` を読み込んで切り替え

### config.json のフォーマット

```json
{
  "shop_name": "武器ショップ",
  "categories": [
    { "id": "abc12345", "name": "銃器", "order": 0 },
    { "id": "def67890", "name": "弾薬", "order": 1 }
  ],
  "products": [
    {
      "id": "a1b2c3d4",
      "name": "ピストル",
      "price": 5000,
      "input_type": "checkbox",
      "category_id": "abc12345",
      "order": 0
    },
    {
      "id": "e5f6g7h8",
      "name": "弾薬（50発）",
      "price": 200,
      "input_type": "quantity",
      "category_id": "def67890",
      "order": 0
    }
  ]
}
```

| input_type | 説明 |
|-----------|------|
| `quantity` | ＋/− で数量を指定（複数個購入向け） |
| `checkbox` | ON/OFF の1点買い向け |

## ❓ よくある質問

**Q. 起動してもショップ名や商品が何も表示されない**  
A. 初回起動時は空の状態です。**[商品設定]** からカテゴリと商品を追加してください。

**Q. 設定を別のPCに移したい**  
A. `config.json` をコピーして、実行ファイルと同じフォルダに置いてください。

**Q. 金額コピーボタンを押しても何も貼り付けられない**  
A. 数値のみがコピーされます（例: `12500`）。`¥` 記号はコピーされません。

**Q. 商品を削除したらカテゴリから消えた**  
A. カテゴリを削除すると、そのカテゴリに属していた商品は「未分類」に移動します。商品自体は削除されません。
